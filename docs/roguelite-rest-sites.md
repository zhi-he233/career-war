# 肉鸽休息点

休息点不一定只是回血，也可以让玩家做构筑选择。

## 可选休息行为

- 回复生命
- 升级一个成长
- 删除一个负面效果
- 获得少量金币
- 强化一个技能
- 跳过休息换奖励

## 休息行为表

| id | 名称 | 效果 | 限制 | 备注 |
| --- | --- | --- | --- | --- |
| heal_at_desk | 趴桌回血 | 回复中量生命 | 每次休息点只能选 1 次 | 最基础选项 |
| polish_skill | 复盘技能 | 强化一个已有技能 | 需要已有可强化技能 | 偏构筑 |
| clear_bad_mood | 调整心态 | 删除一个负面效果 | 没有负面效果时不可选 | 适合中后期 |
| take_side_job | 接个私活 | 获得少量金币 | 不回复生命 | 贪心选项 |
| skip_for_bonus | 不休息继续冲 | 获得一个小奖励 | 不回复生命 | 高风险 |
|  |  |  |  |  |
