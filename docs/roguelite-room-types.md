# 肉鸽房间类型说明

## normal

普通战斗，主线体验，奖励普通。适合放在大多数路线中，让玩家稳定推进。

## elite

高风险战斗，奖励更好，可能给稀有奖励。适合让玩家主动选择风险。

## boss

阶段强敌，不一定严格每 3 关一个，可以作为路线目标或关键节点。Boss 应该有明确主题和更强奖励。

## event

问号事件，不一定战斗，可能有选择和代价。适合写骰子斗技场奇遇、黑市交易、危险试炼、职业残响、风险交换。shop

商店，花金币买东西。可以卖回血、成长、技能、刷新、删除负面效果等。

## rest

休息点，回血 / 强化 / 清负面。适合放在高压路线之后，让玩家恢复或做构筑选择。

## reward

奖励房，不战斗，直接拿奖励或选择奖励。适合做路线收束、Boss 后缓冲、阶段奖励。

## 填写建议

|房间类型|设计重点|常见奖励|常见代价|备注|
|-|-|-|-|-|
|normal|稳定推进|从普通成长池三选一|战斗生命损耗|主线骨架|
|elite|风险收益|从稀有奖励池三选一|敌人生命 +6 伤害 +1（相对普通关）|不要过密|
|boss|阶段目标|从角色技能池或 Boss 能力池三选一|高难战斗，Boss 有独特机制|要有主题|
|event|剧情选择|金币 / 成长 / 回复（具体数值见事件表）|生命 / 负面效果 / 机会成本|文案优先，奖励必须数值化|
|shop|消费选择|商品（见商店表）|金币（消耗）|价格和效果必须数值化|
|rest|恢复强化|回复 10 生命 / 强化 / 删除负面效果|放弃其他选项|路线缓冲|
|reward|直接奖励|从普通成长池三选一 / 20 金币 / 回复 10 生命|通常无|可做爽点|
