# 肉鸽房间类型说明

## normal

普通战斗，主线体验，奖励普通。适合放在大多数路线中，让玩家稳定推进。

## elite

高风险战斗，奖励更好，可能给稀有奖励。适合让玩家主动选择风险。

## boss

阶段强敌，不一定严格每 3 关一个，可以作为路线目标或关键节点。Boss 应该有明确主题和更强奖励。

## event

问号事件，不一定战斗，可能有选择和代价。适合写职场段子、随机机遇、风险交换。

## shop

商店，花金币买东西。可以卖回血、成长、技能、刷新、删除负面效果等。

## rest

休息点，回血 / 强化 / 清负面。适合放在高压路线之后，让玩家恢复或做构筑选择。

## reward

奖励房，不战斗，直接拿奖励或选择奖励。适合做路线收束、Boss 后缓冲、阶段奖励。

## 填写建议

| 房间类型 | 设计重点 | 常见奖励 | 常见代价 | 备注 |
| --- | --- | --- | --- | --- |
| normal | 稳定推进 | 普通奖励 | 生命损耗 | 主线骨架 |
| elite | 风险收益 | 更好奖励 / 稀有奖励 | 更难战斗 | 不要过密 |
| boss | 阶段目标 | 核心奖励 / Boss 能力 | 高难战斗 | 要有主题 |
| event | 剧情选择 | 金币 / 成长 / 回复 | 生命 / 负面 / 机会成本 | 文案优先 |
| shop | 消费选择 | 商品 | 金币 | 价格要清楚 |
| rest | 恢复强化 | 回血 / 强化 / 清负面 | 放弃其他选项 | 路线缓冲 |
| reward | 直接奖励 | 成长 / 技能 / 金币 | 通常无 | 可做爽点 |
