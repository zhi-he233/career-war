# 肉鸽商店

## 商店规则

| 规则项 | 当前建议 | 备注 |
| --- | --- | --- |
| 每次显示几个商品 | 3 | 后续可按阶段增加 |
| 是否可以刷新 | 是 | 刷新消耗金币 |
| 刷新价格 | 20 | 可随刷新次数递增 |
| 是否可以买回血 | 是 | 基础保底商品 |
| 是否可以买技能 | 是 | 中后期更常见 |
| 是否可以买成长 | 是 | 主要消费项 |
| 是否可以删除负面效果 | 是 | 价格应偏高 |
