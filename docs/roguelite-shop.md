# 肉鸽商店

商店主题统一为：地下黑市、铁匠铺、骰子摊、战利品回收站。不要写办公类物品。

## 商店规则

|规则项|当前建议|备注|
|-|-|-|
|每次显示几个商品|3|后续可按阶段增加到 4|
|是否可以刷新|是|刷新消耗金币|
|刷新价格|20|可随刷新次数递增|
|是否可以买回血|是|基础保底商品|
|是否可以买技能|是|中后期更常见|
|是否可以买成长|是|主要消费项|
|是否可以删除负面效果|是|价格应偏高|
|是否可以买路线信息|候选|可用于地图玩法|
|是否可以买临时强化|是|价格低，但只持续下一场|
